# UAIV DataHub

**UAIV DataHub** is the unified gateway for the UAIV low-altitude visual intelligence ecosystem. It connects the public dataset releases, annotation platform, data production infrastructure, and future benchmark resources in one place.

> From low-altitude visual data to annotation, quality control, release, and evaluation.

## Ecosystem Overview

| Asset | Status | Role |
| --- | --- | --- |
| **LAMD-DATASETS** | Released | Public demo release for low-altitude multimodal data |
| **UAIV-Real-CityGov-v1** | In Progress | Flagship city governance dataset under annotation and QA |
| **UAIV-Labeler** | Released | Low-altitude-native annotation and review platform |
| **UAIV-Foundry** | In Progress | Data production, QA, release, and benchmark infrastructure |
| **UAIV-Benchmark** | Planned | Future unified AI-ready benchmark suite |
