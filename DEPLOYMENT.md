# GitHub Pages Deployment

This repository can be published directly through GitHub Pages.

## Files To Upload

Upload these files to the repository root:

```text
index.html
README.md
DEPLOYMENT.md
.nojekyll
```

## Repository Settings

1. Open the GitHub repository.
2. Go to `Settings`.
3. Open `Pages`.
4. Set `Source` to `Deploy from a branch`.
5. Set `Branch` to `main`.
6. Set folder to `/root`.
7. Save the settings.

## Expected URL

```text
https://jennyzhang0810.github.io/UAIV-DataHub/
```

Future updates only require editing `index.html` or `README.md`, then committing and pushing the changes to GitHub.
